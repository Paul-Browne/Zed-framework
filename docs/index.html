Mount

Z.mount({
    key: string, // optional, needed to update component and for data key, eg Z.string
    inner: DOM node, // one of inner or outer needed
    outer: DOM node, // one of inner or outer needed
    before: function(), // optional, fired before the render
    render: function() or string (path to resource), 
    after: function(), // optional, fired after the render
    data: JSON or string (path to resource) // optional, requires key
})

render:

The simplest Z.mount() will just insert the contents on render: into the inner: DOM element.

Z.mount({
    render: function(){
        return "<h1>Hello world!</h1>";
    },
    inner: document.getElementById("insert-here")
})

You can alternatively pass a string to render: which will be interpreted as a path to the HTML resource.

Z.mount({
    render: "path/to/component.html",
    inner: document.getElementById("insert-here")
})

inner: and outer:

inner: and outer: are both DOM node references that are used as the entry point for the render:

The only difference is that the render: will be placed inside the inner: DOM node, but will replace the outer: DOM node.

NOTE: when you use outer: it will not be possible to update that component.

before: and after:

before: and after: are functions that will be executed before and after the render:

Z.mount({
    before: function(){
      console.log("fires before the render");
      return ...
    },    
    render: "path/to/component.html",
    after: function(){
      console.log("fires after the render");
      return ...
    },
    inner: document.getElementById("insert-here")
})

key:

A key: is required if you are updating a component, or if you are associating some data: with that component.

data:

You can also add some data: to the render:, in the following example the array ["bar", "baz", "buff"] will be available globally as Z.foo before the render:

Z.mount({
    key: "foo",
    render: "path/to/component.html",
    inner: document.getElementById("insert-here"),
    data: ["bar", "baz", "buff"]
})

Again, you can pass a string to data: to use an external resource for the data.

Z.mount({
    key: "foo",
    render: "path/to/component.html",
    inner: document.getElementById("insert-here"),
    data: "path/to/data.json"
})

Update

To update a component, with new data, simply do the following. This will re-render the component with the new Z.foo array.

Z.update("foo", {
    data: ["lorem", "ipsum", "dolor"]
})

You can also re-render with a new HTML component, if you so wish.

Z.update("foo", {
    render: "path/to/new-component.html"
})

and you could also pass new before: and/or after: functions.

Z.update("foo", {
    before: null,
    after: function(){
      console.log("after");
    }
})

and you could pass no methods which will just re-render that component.

Z.update("foo")

Listen

You can Z.listen() to a key for when it's updated.

Z.listen("foo", function(){
      console.log("foo updated!");
})

Previous

You can access the prevoius state of a method eg. Z.previous.foo would be the previous state of Z.foo

<!-- index.html -->
<head>
    <script src="js/Z.min.js"></script>
</head>
<body>
	<h1>Counter Demo</h1>
    <button onclick='Z.update("count", {data: Z.count - 1})'>-</button>
    <span id="output"></span>
    <button onclick='Z.update("count", {data: Z.count + 1})'>+</button>
    <script>
        Z.mount({
            key: "count",
            render: "components/counter-total.html",
            inner: document.getElementById("output"),
            data: 0
        })
    </script>
</body>

<!-- counter-total.html -->
<script id="total">
    document.getElementById("total").outerHTML = Z.count;
</script>

Todo List Demo

view live version or edit on glitch
<!-- index.html -->
<head>
    <script src="js/Z.min.js"></script>
</head>
<body>
    <h1>Todo List Demo</h1>
    <div id="todo-list"></div>
    <div id="todo-add"></div>
    <script>
        Z.mount({
            key: "todos",
            render: "components/todo-list.html",
            inner: document.getElementById("todo-list"),
            data: []
        })

        Z.mount({
            render: "components/todo-add.html",
            outer: document.getElementById("todo-add")
        })
    </script>
</body>

<!-- todo-list.html -->
<script id="list-container">
    document.getElementById("list-container").outerHTML = Z.todos.map(function(todo, index){
        return "<div><span>" + todo + "</span><button onclick=removeItem(" + index + ");>remove</button></div>";
    }).join("");

    function removeItem(index) {
        Z.todos.splice(index, 1);
        Z.update("todos", {
            data: Z.todos
        })
    }
</script>

<!-- todo-add.html -->
<input id="task" type="text" placeholder="add task">
<button onclick="addItem()">Add</button>
<script>
    function addItem() {
        var task = document.getElementById("task");
        Z.todos.push(task.value);
        Z.update("todos", {
            data: Z.todos
        })
        task.value = "";
    }
</script>
